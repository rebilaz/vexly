# prismic.io

Prismic est un headless CMS pour développeurs qui permet de construire des sites web performants avec leur stack préféré et de fournir un visual page builder aux marketeurs pour créer rapidement des pages on-brand sans intervention dev. Idéal pour équipes tech et marketing, il supporte croissance via API calls, bande passante CDN, locales multilingues et rôles utilisateurs. Offre illimité documents/types/assets, optimisation images. Plans scalables de Free à Platinum pour PoC, équipes petites à entreprises ajoutant features continuellement. Bénéfices : simplicité, efficacité, scalabilité sans complexité.[1]
